Fluent storage repository with dot notated key support.

## Install

You may install this package using [composer][link-composer].

```shell
$ composer require bhittani/repository
```

## Usage

Usage instructions

```php

<?php

require_once __DIR__ . '/vendor/autoload.php';

// ...

```
